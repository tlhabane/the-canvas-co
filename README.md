# Run and deploy Website

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

1. Run the app:
   `npm run build`
2. Deploy the app:
   `npm run deploy`
